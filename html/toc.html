Overview

DSX Suite is a custom component package from Visual BI for SAP Design Studio consisting of 40+ extensions that includes charts, selectors and other utilities. The suite also offers other features that are not available in the standard set of components. Custom extensions in the suite are compatible across all browsers and mobile devices. The extensions are also tested and available for all the three supported platforms

  • SAP BusinessObjects BI Platform
  • SAP Netweaver BW Platform
  • SAP HANA Platform
Note: For the custom extensions to work on the Design Studio applications, the components need to be installed on the Design Studio client tool and published on the platform server. For more information, please refer to the installation guide.
